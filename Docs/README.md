# Stock Purchase Prototype

Scaffold for the premium stock purchase interaction prototype.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Structure

```
src/
├── app/              # Next.js App Router
├── components/       # UI components
│   └── ui/           # shadcn/ui primitives
├── hooks/            # Custom React hooks
└── lib/              # Utilities and constants
docs/                 # Project documentation
public/               # Static assets
```

## Development

```bash
npm install
npm run dev
```
