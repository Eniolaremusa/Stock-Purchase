# Motion Principles

Every animation should feel calm and premium.

Never use bouncy or exaggerated motion.

Avoid large translations.

Avoid elastic easing.

Use spring motion only where it improves physical realism.

Animation should almost disappear.

If an animation is noticeable, reduce it.
